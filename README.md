API-сервис "Сервис для бронирования переговорок". 
стэк: Python,fastAPI,sqlAlchemy.
В Рамках задания было реализовано 2 модуля:
Авторизация(auth)-позволяет создать или войти в уже существующую учетную запись, система авторизции реализована через JWT. 
Бронирования(booking)-позволяет получить список бронирований, создать бронирование по переданному временному промежутку(от 30 минут до 24 часов) 
с возможностью выбора комнаты, доступны комнаты от 1 до 5 на выбор(предусмотрена проверка уникальности бронирования, не больше 1й брони на промежуток времени и комнату)
